Conclusions and Reflections

I, Krishna Teja Kocherla, contributed to both system design and full-stack implementation of the SpendWise project. I created Domain Class Diagrams (DCDs) and Sequence Diagrams to ensure the architecture followed SOLID and GRASP principles, providing a solid foundation for maintainable code. On the implementation side, I built the Dashboard and Expense Log, implemented the Login functionality, and helped develop the Firestore Singleton class for consistent database access across the app.

The project presented challenges in integrating multi-user features, managing real-time Firebase updates, and maintaining a responsive, user-friendly interface. Overcoming these obstacles enhanced my understanding of full-stack architecture, collaborative development, and practical application of software design principles. Overall, SpendWise provided valuable experience in designing, developing, and delivering a cohesive budgeting application while improving my skills in system architecture, front-end and back-end development, and teamwork.
