User Interface (UI)

The user interface follows a consistent layout supported by a bottom navigation bar that links the main areas of the app. Consistent headers, spacing, and theming help ensure a cohesive experience across all screens and support smooth navigation throughout the app.

Dashboard: The Dashboard presents spending summaries and visual insights using charts, providing users with an at-a-glance view of their financial activity.

Expense Log: The Expense Log screen allows users to add expenses with names, amounts, categories, dates, and optional notes, and displays logged entries in a scrollable list.

Budgets: The Budgets screen enables users to create budget goals, track usage, and view status indicators that reflect progress toward their financial targets.

Savings Circles: Savings Circles provides group savings functionality where users can participate in shared challenges and monitor group contributions toward collective goals.

AI Chatbot: The AI Chatbot offers budgeting tips and replies to user questions within a clean message interface, providing helpful guidance and financial advice.
